Frequently Asked Questions

How do I add something to Werehouse?

There are three ways to add something to your archive in Werehouse currently:

  • Web Form
    Go to the Add to Queue page and submit a link or a file.
  • Telegram Bot
    If you haven’t already, send /start to @werehouse_archive_bot on Telegram and click the link it sends you. This will link your Telegram account with Werehouse. Then, you can forward messages to that bot to add links and images to your queue.
  • Manual Upload
    Go to the Archive page and submit an image file.

Soon, Werehouse will support manually archiving videos too. I have yet to figure out how I want to create thumbnails from videos, which is why it isn’t supported already.

How do I invite someone?

To invite someone to use this instance of Werehouse, follow these steps:

  1. Send a message to whoever runs this instance and ask them to “add an invite link to your account.”
  2. Go to your account page and scroll down to the Invite Links section.
  3. You should see that you have 1 invite link that doesn’t say “Used.” Copy that link and send it to the person you’d like to invite.
Why does Werehouse only support Mastodon servers, not other Fediverse server software?

Mastodon offers a client API that is easy to integrate into Werehouse and requires no authorization. However, if Werehouse wants to directly speak ActivityPub, it basically needs to become an ActivityPub server! This is because of features like Secure Mode in Mastodon and the default configuration for GoToSocial. These features require that Werehouse provide a valid ActivityPub server signature on its requests, which in turn requires creating an “instance actor” to allow other ActivityPub servers to find its public key.

Werehouse will eventually support ActivityPub properly, after I’ve had sufficient time to develop, test, and try out the ActivityPub code.
