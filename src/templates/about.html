A Personal Art Archiving Tool

Hyperphantasia is a tool for keeping track of the art you care about and want to be able to find again. It lets you store reference images, art you want to save for later, and art you want to share with friends, and it makes it easy to keep track of who created each artwork.

Built for Friends

I'm not trying to sell this or “productize” it. Hyperphantasia is open source and permissively licensed, and designed to be used by yourself or with a small group of friends. Archive the art you care about so that you’ll always have access to it!

Minimalism is a Feature

This kind of service should focus on the artwork, not load down your browser with megabytes of bloated JavaScript. All the basic functionality of Hyperphantasia works with plain old HTML 5, and the advanced features use JavaScript sparingly. That said, I’m perfectly willing to use JavaScript when it improves the user experience!
