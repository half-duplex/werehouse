A Personal Art Archiving Tool

Werehouse is a tool for keeping track of the art you care about and want to be able to find again. It lets you store reference images, art you want to save for later, and art you want to share with friends, and it makes it easy to keep track of who created each artwork.

Built Small

I'm not trying to sell this or “productize” it. Werehouse is open source and permissively licensed, and designed to be used by yourself or with a small group of friends. Archive the art you care about so that you’ll always have access to it!

Minimalism is a Feature

This kind of service should focus on the artwork, not load down your browser with megabytes of bloated JavaScript. All the basic functionality of Werehouse works with JavaScript disabled, and the advanced features use JavaScript judiciously.
